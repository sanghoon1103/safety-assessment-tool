농업 안전관리 프로그램 통합 사용자 매뉴얼

본 문서는 '위험성 평가 프로그램'과 '안전관리 수준 평가 프로그램'의 사용법을 안내하기 위해 제작되었습니다. 각 프로그램의 목적과 단계별 사용법을 숙지하여 농작업 환경의 안전성을 체계적으로 관리하시기 바랍니다.

1. 위험성 평가 프로그램 사용자 매뉴얼

프로그램 소개

이 프로그램은 농작업 현장에서 발생할 수 있는 다양한 위험 요소를 체계적으로 파악하고, 개선 대책을 수립하여 위험도를 낮추기 위해 설계된 디지털 위험성 평가 도구입니다. 사용자는 미리 입력된 농작업별 위험 시나리오를 선택하거나, 현장에 맞는 위험 요인을 직접 입력하여 평가를 진행할 수 있습니다.

단계별 사용법

1단계: 기본 정보 입력

프로그램 상단의 기본 정보 섹션에 평가자명, 전화번호, 지역명, 작목반을 입력합니다. 이 정보는 보고서 생성 및 파일 저장 시 사용됩니다.

2단계: 위험성 평가 실행 (개선 전)

  1. 행 추가/삭제: [행 추가] 버튼으로 평가 항목을 추가하고, 각 행의 휴지통 아이콘으로 삭제합니다.
  2. 작업 선택: 평가하려는 작업(예: 수확)과 세부작업(예: 과수 수확)을 목록에서 선택합니다.
  3. 위험 선택: [위험 선택] 버튼을 눌러 나타나는 팝업 창에서 농장 상황과 가장 유사한 위험 사례를 선택합니다. 필요시 '실제 문제점' 내용을 수정하고 가중치를 조절한 후 [선택 완료]를 누릅니다.
  4. 직접 입력: 목록에 없는 위험은 '작업 선택'에서 [직접 입력]을 선택하여 작업명, 문제점, 위험도(강도, 빈도, 가중치)를 수동으로 입력할 수 있습니다.

3단계: 개선 대책 수립 및 평가 (개선 후)

  1. 대책 입력: '3. 개선 후 위험성 평가' 표의 '대책' 칸에 위험을 줄이기 위한 구체적인 실천 계획을 작성합니다.
  2. 개선 후 위험도 입력: 수립한 대책이 실행되었다고 가정했을 때, 예상되는 '개선 후 강도', '빈도', '가중치'를 각각 입력합니다. '개선 후 위험도'와 '위험성 감소율'은 자동으로 계산됩니다.

4단계: 결과 확인 및 저장

  1. 종합 결과 확인: 모든 평가 완료 후, 하단의 '종합 평가 요약' 섹션에서 개선 전/후 위험도 총점과 총 감소율을 한눈에 파악할 수 있습니다.
  2. 보고서 생성: [보고서 생성] 버튼을 누르면 인쇄 가능한 형태의 깔끔한 보고서가 새 탭으로 열립니다.
  3. CSV 저장: [CSV 저장] 버튼을 누르면 모든 평가 데이터가 담긴 엑셀 호환 파일(.csv)이 지역명_작목반명으로 자동 저장됩니다.

주요 기능 및 팁

  • 새 평가 시작: 여러 사람이 같은 컴퓨터를 사용할 경우, [새 평가 시작 (기존 내용 삭제)] 버튼을 눌러 이전 사용자의 기록을 지우고 깨끗한 상태에서 시작할 수 있습니다.
  • 자동 저장: 모든 평가 내용은 브라우저에 실시간으로 자동 저장되어, 실수로 창을 닫아도 작업 내용이 유실되지 않습니다.

2. 안전관리 수준 평가 프로그램 사용자 매뉴얼

프로그램 소개

이 프로그램은 농작업 현장의 안전관리 수준을 정량적으로 평가하고, 개선 계획에 따른 기대 효과를 예측하기 위해 개발된 전문 진단 도구입니다. 5가지 핵심 분야의 세부 지표에 대해 개선 전과 후의 수준을 각각 평가하여, 안전관리 수준이 얼마나 향상될 수 있는지를 증가율(%)로 한눈에 파악할 수 있습니다.

단계별 사용법

1단계: 기본 정보 입력

프로그램 상단의 기본 정보 섹션에 평가자명, 전화번호, 지역, 작목반을 입력합니다. 이 정보는 최종 보고서에 반영됩니다.

2단계: 평가 진행

  1. 평가지표 선택: 메인 화면의 지표 목록에서 평가하려는 항목(예: 작업 자세 개선...)을 클릭하여 상세 평가 페이지로 이동합니다. 각 지표의 아이콘으로 평가 진행 상태를 확인할 수 있습니다.
  2. 등급 선택: 상세 페이지에서 '개선전 수준'과 '개선후 수준'에 대해 각각 A~E 등급 버튼 중 하나를 선택합니다. 페이지 하단의 '평가 기준'을 참고하여 객관적으로 평가합니다.
  3. 상태 기록: '개선전 상태'와 '개선후 상태' 입력란에 현재 상황과 개선 목표를 구체적으로 서술합니다. 이 내용은 최종 보고서에 포함됩니다.
  4. 목록으로 복귀: ← 목록으로 돌아가기 버튼을 눌러 메인 화면으로 돌아온 후, 다른 모든 지표에 대해 평가를 반복합니다.

3단계: 결과 확인 및 활용

  1. 종합 결과 확인: 페이지 하단에서 개선전/후 총 점수와 '안전관리 수준 증가율(%)'을 즉시 확인할 수 있습니다.
  2. 시각화 차트: 막대그래프를 통해 개선전, 개선후, 총 만점 점수를 직관적으로 비교할 수 있습니다.
  3. 보고서 생성: [보고서 생성] 버튼을 누르면 모든 평가 내용이 담긴 전문 보고서가 새 탭으로 열립니다. 인쇄하거나 PDF로 저장할 수 있습니다. (팝업 차단 해제 필요)
  4. CSV로 내보내기: [CSV로 내보내기] 버튼으로 모든 평가 데이터를 엑셀 호환 파일(.csv)로 다운로드하여 별도로 관리하거나 분석할 수 있습니다.
